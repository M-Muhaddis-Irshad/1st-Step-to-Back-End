# 🚀 1st-Step-to-Back-End

A beginner-friendly backend project built using Express.js and MongoDB.

## 📌 Project Overview
This project demonstrates basic backend development concepts:
- Express server setup
- REST API basics
- MongoDB integration
- Middleware usage

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- JavaScript (ES6+)

## ⚙️ Installation

git clone https://github.com/your-username/1st-Step-to-Back-End.git
cd 1st-Step-to-Back-End
npm install
node index.js

## 🚀 Features
- Express server
- API routes
- MongoDB connection
- JSON handling
